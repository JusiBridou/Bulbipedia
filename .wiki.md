# Project Summary
Bulbipédia is a collaborative Pokémon encyclopedia inspired by Bulbasaur, designed to provide comprehensive information about Pokémon, their stats, evolutions, and more. The project aims to create a user-friendly platform where fans can explore and contribute to a growing database of Pokémon-related knowledge, similar to Wikipedia.

# Project Module Description
The project consists of several functional modules:
- **Wiki Header**: Displays the site logo, navigation links, and a search bar.
- **Wiki Sidebar**: Provides navigation options for Pokémon types and recent activities.
- **Wiki Infobox**: Displays key statistics and information about individual Pokémon.
- **Wiki Article**: Renders detailed articles about Pokémon, including sections for etymology, physiology, and statistics.
- **Home Page**: Features a hero section, popular articles, and a search functionality.
- **Category Page**: Allows users to explore Pokémon by type and generation.
- **Search Page**: Displays search results based on user queries.
- **Not Found Page**: Shows a friendly message when a user navigates to a non-existent page.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for modern web applications.
- **React Router**: For routing and navigation within the app.
- **TanStack Query**: For data fetching and state management.

# Usage
1. **Install Dependencies**: Run `pnpm install` to install project dependencies.
2. **Build the Project**: Use `pnpm run build` to compile the application.
3. **Run the Development Server**: Use `pnpm run dev` to start the server for local development.
